This is a movie searching app based on the Open Movie Database. 
Users can search movies by keywords and star their favorite movies.


Creative portion - a “settings” tab with three additional functions!

	1. Filter results that do not have posters
		If OMDb does not have a link for the poster, 
		then the movie does not display in results.

	2. Filter results whose types are not movies
		If OMDb records a movie of type “series” or “episode, ”
		then the movie does not display in results.

	3. Specify the number of rows displayed
		By default, the app shows 10 rows in every search.
		Can be any value within 100 but may cause longer wait.
